Image Monitoring - SURVEILLANCE 2.0

Problem statement:

The current platform of GECS doesn't monitor images sent through different communication mediums in HSBC. We aim to create a tool to handle this issue. This tool can be used to convert images to text which can further be used to trigger breached policies., which were being missed out by the current platform.


What we offer!

Image monitoring platform provides the utility to flag an image based on the GECS policies. The tool parses through images, extracting the words mentioned in and cross verifying the content against the policy data.


Future scope:

  • Add Apache Spark: Will enable us for large scale data processing.
  • Text monitoring for ISR team
  • NLP and Text Mining
