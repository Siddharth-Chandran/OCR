Image to Text - SURVEILLANCE 2.0

The current platform of GECS doesn't monitor images sent through different communication mediums in HSBC. We aim to create a tool to handle this issue. This tool can be used to convert images to text which can further be used to trigger breached policies., which were being missed out by the current platform.

Technologies and Frameworks

SpringBoot
ThymeLeaf
Tess4j (Java wrapper for Google Tesseract)
Tomcat
Maven
Java 8
HTML 5 + JQuery
